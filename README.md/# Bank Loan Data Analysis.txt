# Bank Loan Data Analysis

## 📊 Overview
This project explores 38,600+ loan applications to uncover key trends, risk factors, and financial insights using **Microsoft Excel**. The analysis helps identify high-risk customers and optimize lending decisions.

## 🔧 Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Conditional Formatting)
- Dashboarding techniques

## 🧠 Key Features
- Visual analysis by **loan grade, purpose, state, and employment length**
- Separation of **Good vs Bad loans**
- KPIs: Average interest rate, funded amount, loan term, debt-to-income ratio (DTI)
- Drill-down filtering for dynamic insights

## 📌 Insights
- Identified high-risk loan segments based on employment length and homeownership
- Certain states show higher loan defaults despite good DTI
- Top 3 purposes of loan with the highest return rate were medical, debt consolidation, and small business
